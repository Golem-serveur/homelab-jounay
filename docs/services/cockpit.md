# Cockpit

## Objectif

Administrer mon serveur Debian via une interface web.

## Ce que j'ai appris

- Gestion des services.
- Surveillance des ressources.
- Administration système via le navigateur.

## Fonctionnalités utilisées

- Vue d'ensemble du système.
- Utilisation CPU et RAM.
- Gestion des services.
- Terminal web.

## Commandes utilisées

```bash
sudo apt install cockpit

sudo systemctl enable --now cockpit
```

## Difficultés rencontrées

- Comprendre les fonctionnalités disponibles.

## Notes personnelles

Cockpit est une excellente interface pour débuter l'administration système.
