# Java

## Objectif

Installer Java pour exécuter mon serveur Minecraft.

## Commandes utilisées

```bash
java --version

sudo apt install default-jdk
```

## Ce que j'ai appris

- Vérifier la version de Java.
- Installer un JDK.
- Comprendre la dépendance entre Java et Minecraft.

## Difficultés rencontrées

- Comprendre la différence entre JRE et JDK.

## Notes personnelles

Java est indispensable pour héberger un serveur Minecraft.
