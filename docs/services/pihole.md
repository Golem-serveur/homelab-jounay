# Pi-hole

## Objectif

Bloquer les publicités et analyser les requêtes DNS sur mon réseau local.

## Fonctionnalités

- Blocage des publicités.
- Statistiques DNS.
- Gestion des listes de blocage.
- Interface web.

## Commandes utilisées

```bash
curl -sSL https://install.pi-hole.net | bash
```

## Ce que j'ai appris

- Fonctionnement du DNS.
- Filtrage des domaines.
- Configuration d'un serveur DNS local.

## Difficultés rencontrées

- Comprendre le rôle du DNS.
- Configuration du réseau.

## Notes personnelles

Pi-hole est l'un des services les plus utiles de mon Homelab. Il me permet de mieux comprendre le fonctionnement des requêtes DNS et d'améliorer mon confort de navigation.
