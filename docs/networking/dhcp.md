# DHCP

## Objectif

Comprendre comment les appareils obtiennent automatiquement une adresse IP.

## Ce que j'ai appris

- Attribution automatique des IP.
- Rôle du routeur.
- Réservation d'adresses IP.

## Exemple

PC -> Demande une IP
Routeur -> Attribue une IP

## Notes personnelles

Le DHCP sera particulièrement important lorsque je mettrai en place des VLAN et OPNsense.
