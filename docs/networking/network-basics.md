# Network Basics

## Objectif

Comprendre les bases du réseau pour mon Homelab.

## Ce que j'ai appris

- Différence entre Internet et réseau local.
- Fonctionnement d'une adresse IP.
- Ports réseau.
- TCP et UDP.

## Concepts importants

- LAN
- WAN
- IP privée
- IP publique
- Routeur
- Switch

## Commandes utilisées

ip a

ping 8.8.8.8

ss -tulnp

hostname -I

## Difficultés rencontrées

- Comprendre les ports.
- Différence entre IP publique et privée.

## Notes personnelles

Comprendre les bases du réseau est indispensable avant d'aller plus loin (Docker, Proxmox, OPNsense, VLAN).
