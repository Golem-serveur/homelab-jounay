# Lynis

## Objectif

Auditer la sécurité de mon système Debian.

## Commandes utilisées

```bash
sudo apt install lynis
sudo lynis audit system
```

## Ce que j'ai appris

- Audit système.
- Hardening.
- Bonnes pratiques de sécurité.

## Difficultés rencontrées

- Comprendre certains avertissements.

## Ressources

- https://cisofy.com/lynis/

## Notes personnelles

Lynis permet de voir rapidement les points faibles d'un système.
