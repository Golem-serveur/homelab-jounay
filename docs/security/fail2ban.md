# Fail2ban

## Objectif

Protéger les services contre les tentatives de connexion répétées.

## Commandes utilisées

```bash
sudo apt install fail2ban
sudo systemctl status fail2ban
sudo systemctl enable fail2ban
```

## Ce que j'ai appris

- Protection contre le brute force.
- Importance des logs.
- Sécurisation de SSH.

## Difficultés rencontrées

- Comprendre le fonctionnement des jails.

## Ressources

- https://www.fail2ban.org/
