# Nmap

## Objectif

Découvrir les services et les ports ouverts sur mon réseau.

## Commandes utilisées

```bash
nmap 192.168.x.x

nmap -sV 192.168.x.x

nmap -p 1-1000 192.168.x.x
```

## Ce que j'ai appris

- Fonctionnement des ports.
- Découverte réseau.
- Identification des services.

## Difficultés rencontrées

- Comprendre les différents types de scans.

## Ressources
